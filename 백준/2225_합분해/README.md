3개의 정수로 3을 만드는 경우는 다음으로 나타낼 수 있다.

0~3의수 k개로 n을 나타내는 경우의 수를 dp[k][n]이라고 하면

dp[3][3] =>

1. dp[2][0] + 3
   00 + 3
2. dp[2][1] + 2
   01 + 2
   10 + 2
3. dp[2][2] + 1
   20 + 1
   02 + 1
   11 + 1
4. dp[2][3] + 0
   30 + 0
   03 + 0
   12 + 0
   21 + 0

이렇게 일반화 할 수 있다.

dp[k][n] = dp[k - 1][n - n] + dp[k - 1][n - (n - 1)] + ... + dp[k - 1][n - 1] + dp[k - 1][n]
