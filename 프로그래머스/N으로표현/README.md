5를 4번 쓰는 경우를 생각해보자.

`(i개짜리)사칙연산(4-i개짜리)`로 나타낼수있다.

예를 들어
5/((5*5)/5) 은 (1)연산(3)인 경우,
(5/5)+(5*5) 은 (2)연산(2)인 경우,
((5-5)+5)+5 은 (3)연산(1)인 경우이다.
