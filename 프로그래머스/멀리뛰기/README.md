1. `3을 만들 수 있는 조합` + 1
2. `2를 만들 수 있는 조합` + 2
   로 생각해 볼 수 있다.

**n = 3일때 경우의 수** + 1
**1 + 1 + 1** + 1
**1 + 2** + 1
**2 + 1** + 1

**n = 2일떄 경우의 수** + 2
**1 + 1** + 2
**2** + 2

n = 4일 때 총 5개의 조합이 나온다.
